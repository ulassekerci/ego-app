## EGO API

### AUTH

Even though there is no login-signup there is an authentication system in the API. Every request needs a UID (user id) parameter. You get a userID from androidConnect.aspx endpoint when app is first run and use that same userID afterwards. But that userID has a finite lifetime (the exact duration is unknown/untested) and then you need to renew it. The official app sends a request to androidConnect.aspx endpoint with its userID everytime you open the app because sending the same userID as UID param with the same request renews that userID's life. Our app should also do the same, hit the androidConnect endpoint when app runs to either acquire a userID (first run) or to renew the userID it already has.

### Headers

Every request should have these headers:

- User Agent: A user agent header in right format is mandatory to get a data in return. I don't know how the user agent is formatted in IOS app so the best option is to send an android user agent:`user-agent: EGO Genel Mudurlugu-EGO Cepte-ANDROID-samsung-SM-S942B-osV:16.0.0-apV:-`
- accept-encoding: gzip
- accept: application/json; charset=UTF-8

### Notes:

- In `table_saat` rows (line details), every departure has a `detay` field but it's `"-"` (or empty) when there is no actual note — treat those as no detail.
- API returns objects that behave differently based on whether a bus is actively on the road or not, :
  - No bus: arac_no is "-" and sure contains a long newline-separated string ("Sonraki Hareket Saati...").
  - Active bus: arac_no has an ID, and fields like plaka_no, saniye, and detay appear.
- Active buses at or just past the selected stop get special sure/durum values. saniye is always "0" in these states, so check sure/durum:
  - Bus is at the stop right now: `"sure":"Geldi"` and `"durum":"geldi"`, e.g. `{"arac_no":"07-157","plaka_no":"06 BG 3495","lat":"39.893074","lng":"32.83469","hiz":"49 km","durak_no":"11108","onceki_durak_no":"11107","hat_no":"155-6","konum_tarihi":"09.07.2026 01:38:39","durak_sira_no":"12","aci":"172","detay":"Solo‚ Engelli","saniye":"0","sure":"Geldi","secili_durak_no":"11108","secili_durak_sira_no":"12","yon":"4","durum":"geldi","konum":"-","hat_ad":"ATATÜRK SİTESİ-SOKULLU CD.-ULUS"}`.
  - Bus just left the stop: `"sure":"Gidiyor"` and `"durum":"gidiyor"`, e.g. `{"arac_no":"37-082","plaka_no":"06 HO 1226","lat":"39.917065","lng":"32.854259","hiz":"35 km","durak_no":"12207","onceki_durak_no":"11612","hat_no":"263-7","konum_tarihi":"09.07.2026 12:19:42","durak_sira_no":"29","aci":"180","detay":"-","saniye":"0","sure":"Gidiyor","secili_durak_no":"12207","secili_durak_sira_no":"29","yon":"4","durum":"gidiyor","konum":"-","hat_kod":"263-7","hat_ad":"(ÖHO) ETLİK-AŞAĞI EĞLENCE-BAKANLIK-BALGAT"}`.
